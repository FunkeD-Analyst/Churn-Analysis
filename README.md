
# 📊 Customer Churn Analysis 

## 🔍 Project Overview

This project focuses on analyzing customer churn behavior to identify key factors influencing customer attrition. The objective is to generate actionable insights that can help organizations improve customer retention and reduce revenue loss.

The analysis was carried out using Microsoft Excel, leveraging data cleaning, statistical techniques, and interactive dashboard design.

## 🎯 Objectives

* Analyze customer churn patterns
* Identify key drivers of customer attrition
* Evaluate the impact of customer behavior on churn
* Provide data-driven recommendations for retention strategies
  
## 🛠️ Tools & Techniques

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Feature Engineering (Churn Flag, Tenure Groups)
* Outlier Detection using Interquartile Range (IQR)
* KPI Development & Dashboard Design

## 📈 Key Insights

* The overall churn rate is approximately 33%, indicating a significant retention challenge.
* Customers with shorter tenure are more likely to churn, highlighting early-stage dissatisfaction.
* Month-to-month contract customers show the highest churn rate compared to long-term contracts.
* Customers with higher monthly charges tend to churn more frequently, suggesting price sensitivity.
* Payment methods requiring manual processing show slightly higher churn patterns.

## 💡 Recommendations

* Improve customer onboarding to reduce early churn
* Encourage long-term contracts through incentives
* Optimize pricing strategies to improve perceived value
* Promote automated payment methods for better convenience

## 📊 Dashboard Features

* Interactive filters (Slicers)
* KPI Cards (Total Customers, Churn Rate, Average Charges)
* Churn distribution visualization
* Behavioral analysis by tenure, contract type, and payment method

## 🚀 Business Impact

This analysis demonstrates how data-driven insights can support strategic decisions aimed at improving customer retention, reducing churn, and enhancing long-term business performance.

## 📌 Conclusion

The project highlights the importance of understanding customer behavior through data analysis. By addressing key churn drivers, organizations can take proactive steps to improve customer satisfaction and retention.
